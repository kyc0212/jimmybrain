英倫風互動搶答平台｜給社區活動、教學帶領、投影與手機顯示使用

British-style quiz experience

像 Kahoot 的流程，套進你自己的風格

這份版本以 GitHub Pages 直接可跑的單檔 `index.html` 為設計核心，重點放在活動現場可用、投影畫面夠大、控制台夠清楚、題庫可自己編。

1
大螢幕題目展示
題目字體大、四個彩色選項塊、圖片題、倒數時間圈。
2
加入流程頁面
輸入 PIN、輸入名稱、玩家清單即時顯示，適合現場報隊名。
3
主持人控制台
開始、揭曉答案、下一題、重設活動、檢視玩家與排行。
4
題目編輯器
可在頁面直接新增題目，也能匯入或匯出 JSON 題庫。
先講清楚：這版已經很接近 Kahoot 的畫面與流程，但因為你要求的是 單一 `index.html` 上 GitHub，所以目前屬於單機正式版，不是跨手機即時同步版。要真的做到每台手機同步搶答，只差下一步加 Firebase。
